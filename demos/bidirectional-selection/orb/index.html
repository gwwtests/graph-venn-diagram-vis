@memgraph/orb — HTTP Server Required

This visualization cannot run from file:// URLs.

Why is the page blank?

Orb uses Web Workers via import.meta for its rendering pipeline. Browsers block Web Worker creation from file:// due to CORS / same-origin restrictions. All other demo visualizations work from file:// — only orb has this limitation.

How to view this demo

Serve the demos directory with any HTTP server:

# Option A — Python
python3 -m http.server 8234 -d demos

# Option B — Node / npx
npx serve demos -l 8234

Then open: http://localhost:8234/bidirectional-selection/orb/

Or use the helper script from the repo root: ./serve-demos.sh orb

About Orb

@memgraph/orb is a graph visualization library by Memgraph, using WebGL for high-performance rendering of large graphs.
